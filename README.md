## Fitness Helper

This Web-site helps Users to pick exercises and create their own workouts. User needs to sign-up to keep tracking workouts or favorite exercises in Database. Exercises has categories , which makes this app easy to use even for beginners. Simple to use, but very helpful App.

## Instructions how to use this app:

1. Homepage has menu: workouts/exercises/nutrition/login
2. User has access to exercises without login
3. To create workouts User needs to Sign-in/Sign-up
4. To Sign-up User needs to fill: A. Required: email, password, name. B. Optional: age and weight
5. After Sign-in/Sign-up Users name will be on top right corner. To logout click on "name"
6. To create Workouts click on "add", then create the name of workout
7. To add exercises click on name of workout, then click "add exercises"
8. User can delete unwanted exercises
9. Nutrition - WIP
